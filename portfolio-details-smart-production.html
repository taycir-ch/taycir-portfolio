Smart Production Platform — IoT Production Monitoring & OEE Analytics

Project Overview & Engineering Challenges

The Smart Production Platform is an intelligent industrial monitoring system designed to digitalize and optimize production processes within a jeans manufacturing environment. The project combines Internet of Things (IoT), artificial intelligence, real-time communication, and production analytics to automate data collection and improve operational efficiency.

The system enables workers to authenticate using RFID badges before interacting with the production line. Manufactured items are automatically analyzed through connected camera devices capable of identifying quality defects using computer vision models. Production data is then processed and transmitted to a centralized platform providing managers with real-time visibility into production activities and performance indicators.

IoT Architecture & Real-Time Communication

The system architecture follows a distributed IoT model integrating ESP32 devices and Raspberry Pi edge-computing nodes responsible for collecting and processing production events. MQTT was implemented as the communication protocol between hardware components and backend services, while WebSockets ensured real-time synchronization with the dashboard interface.

AI Quality Inspection & Production Analytics

Computer vision capabilities were integrated using Python and YOLO object detection models to automatically identify defective products during production. Performance metrics were continuously analyzed through intelligent dashboards capable of calculating Overall Equipment Effectiveness (OEE), monitoring operator productivity, and generating actionable insights for production managers.

System Design & Deployment

The backend infrastructure was developed using NestJS and PostgreSQL with JWT-based authentication mechanisms for secure access control. Docker containers were used for deployment and service orchestration, while the platform was deployed on Oracle Cloud VPS, providing a scalable environment capable of handling industrial data streams and real-time workloads.
